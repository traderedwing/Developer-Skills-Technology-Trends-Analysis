# Developer Skills & Technology Trends Analysis

## Project Overview

This project analyzes global developer ecosystem trends using data collected from multiple sources, including the Stack Overflow Developer Survey, APIs, web scraping, and structured datasets.

The goal of the project was to identify the most in-demand programming skills, technologies, databases, and development tools in today's technology market.

The project covers the complete data analytics workflow — from data collection and cleaning to exploratory analysis, dashboard development, and business storytelling.

---

## Business Problem

Technology evolves rapidly, making it difficult for companies and professionals to understand which technical skills are becoming the most valuable.

This project answers key business questions such as:

- Which programming languages are currently the most demanded?
- Which database technologies dominate the market?
- Which IDEs and developer tools are the most popular?
- What technology trends can support hiring and learning decisions?
